## Quick Start

### Build and Run

```powershell
# Build C++ Backend
cd D:\Projects-Personal\ClipVault2
.\build.ps1

# Build Electron UI
cd ui
npm run build:react
npm run build:electron

# Full build (both)
npm run build

# Run standalone backend (tray icon only)
.\bin\ClipVault.exe

# Run full app (UI + backend auto-start)
.\ui\release\win-unpacked\ClipVault.exe
```

### Usage

1. **Double-click** `ClipVault.exe` (packaged version)
2. **Backend** starts automatically (shows in system tray with custom icon)
3. **UI** opens showing clip library
4. **F9** - Press to save clip anytime (works even if UI is closed)
5. **Click X** on UI - Minimizes to tray, backend keeps recording
6. **Right-click tray** - Open / Open Clips Folder / Exit

### Features

**Recording**
- 120-second replay buffer (always recording last 2 minutes)
- F9 hotkey saves clip instantly
- 1080p60, NVENC hardware encoding
- Two audio tracks: desktop + microphone

**Clip Library**
- Browse all saved clips
- Thumbnails generated automatically
- Search, filter, sort
- Tags and favorites

**Editor**
- Trim clips (non-destructive)
- Adjust audio track volumes
- Export with FFmpeg
- Drag-and-drop to Discord/files
